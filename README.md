# NodeJS_Express_CRUD_MySql

## Dependecies :

* Node JS ```v18.14```
* MySql ```v8.0```

## Tools :

* Visual Studio Code
* XAMPP
* DBeaver

## Installation and Running

### Manual Installation and Running

1. Run this command to Install dependencies

```
npm init
npm install express mysql body-parser
npm install multer
npm install path
```

2. Run this command

```
node index.js
```
### Docker

1. Run this command
```
docker-compose up -d
```
